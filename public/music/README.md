# Music for the cassette player

Three public-domain instrumental recordings, all sourced from the Internet Archive.

| File         | Piece                              | Source                                                                                                              |
|--------------|-------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| `track1.mp3` | Debussy — Clair de Lune             | [archive.org/details/claude-debussy-clair-de-lune](https://archive.org/details/claude-debussy-clair-de-lune)        |
| `track2.mp3` | Satie — Gymnopédie No. 1            | [archive.org/details/erik-satie-gymnopedie-no.-1_music](https://archive.org/details/erik-satie-gymnopedie-no.-1_music) |
| `track3.mp3` | Pachelbel — Canon in D              | [archive.org/details/canon-in-d-pachelbel](https://archive.org/details/canon-in-d-pachelbel)                        |

All three compositions are public domain (composers died well before the 1928 cutoff).
The specific recordings linked above are also public domain or community-uploaded on
Internet Archive and free to redistribute.

## Replacing or adding tracks

1. Drop a new `.mp3` into this folder using one of the filenames above (or add new ones).
2. Update the `tracks` array at the top of `src/components/CassettePlayer.astro` so the
   on-screen labels match.
